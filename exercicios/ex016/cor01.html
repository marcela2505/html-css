Cores em CSS

Para entender esse exercício, olhe bem atentamente o código fonte desse arquivo.
Abaixo temos 4 formas de representar cores em CSS:

Usando cores por nome

Usando cores pelo código hexadecimal

Usando códigos RGB(red, green, blue)

Usando códigos HSL(hue, saturation, light)

As cores resultantes são exatamente as mesmas, mas o formato de representação muda conforme o padrão escolhido.

Também é possível usar transparência, usando o formato RGBA(red, green, blue, alpha), HSLA ou código hexadecimal com dois dígitos a mais.

Nesse título o fundo azul teve sua transparência configurada para 50%
